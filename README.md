# Git Mini Projects
This repository contains hands-on **Git-Focused mini projects** designed to demonstrate real-world usage of Git and GitHub.

These projects focus on **practical workflow**, not theory, and how Git is actually used in individual and team environments.

---

## Project Included

### Project -1 : Git Workflow Simulation
**Focus: Core Git-Mastery**

- Repository initialization
- Branch creation and management
- commits and commits history
- Merging branches
- Handling merge conflicts
- Reset, revert, and recovery

Purpose:
>To demonstrate complete understanding of day-to-day Git workflow

---

### Project -2 : Collaboration & Pull Request Workflow
**Focus: Team-style GitHub usage**

- Feature branches
- Pull Requests (PR)
- Code Review Flow
- Protected branches
- Forking and upstream sync
- Issue-based development

Purpose:
>To simulate how Git is used in teams and open-source projects
