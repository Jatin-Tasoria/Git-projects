# Collaboration & Pull Request Workflow Project

This project demonstrates how Git and GitHub are used in real-world
team and open-source collaboration environments.

## Concepts Demonstrated
- Feature branch workflow
- Pull Requests (PR)
- Code review process
- Protected main branch
- Forking repositories
- Upstream and origin usage
- Issue-based development

## Purpose
To simulate professional GitHub collaboration practices used in
software development teams and open-source projects.

## Tools
- Git
- GitHub
